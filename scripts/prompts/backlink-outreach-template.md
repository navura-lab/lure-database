リンクを直接依頼しない。データ協力の関係構築が目的。

## 運用ガイドライン

### 送信優先度
1. **Tier 1（最優先）**: 商品数50+のメーカー（jackall, daiwa, shimano, megabass等）— ブランド力が高く、被リンク効果大
2. **Tier 2**: 商品数20-49のメーカー — 中堅メーカー、返信率が高い傾向
3. **Tier 3**: 商品数20未満 — 個人経営に近い会社が多く、対応してもらいやすい

### 送信ペース
- 週5社ペース（1日1社）
- フォローアップは初回から7日後
- 同一メーカーへの連絡は最大3回まで

### NGルール
- リンク依頼を直接しない（テンプレートCのお礼時のみ、控えめに言及可）
- 競合他社の名前を出さない
- 「SEO」「被リンク」「ドメインパワー」等の用語を使わない
- 嘘のデータ（実際より多い商品数等）を書かない

### カスタマイズのポイント
- メーカーの最新製品や話題の製品に触れると返信率が上がる
- 小規模メーカーほど「自社製品がデータベースに載っている」ことを喜ぶ傾向
- ソルトメーカーには「対象魚種別の検索機能」を訴求
- バスメーカーには「カラー一覧の充実度」を訴求
- USメーカーには英語テンプレートを別途用意する必要あり（本テンプレートは日本語メーカー向け）

### 成果トラッキング
送信・返信・被リンク獲得を記録するスプレッドシートを別途用意する。
記録項目: メーカー名 / 送信日 / テンプレート種別 / 返信有無 / 返信日 / 被リンク獲得 / URL / 備考
